# SIMPLE SUDOKU GAME
------------------------------------

Built with React, Redux, Webpack.

Demo - https://yanasavchenko.github.io/sudoku/

## HOW TO START
1. npm install
3. npm run devserver
3. open http://localhost:8090 in browser

## CREATE BUILD
1. npm install
2. npm run build
3. root folder contains production version. Just open index.html in browser
